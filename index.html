게임 가이드

🎮 게임 모드
랜덤 10문제 — MC/OX/주관식 균등 배분, 랭킹 미반영
전체 풀기 — 선택 섹션 전체 문제, 랭킹 반영
타임어택 — 문제당 10초 제한, 배점 x2 + 타임 보너스
오답노트 — 틀린 문제만 모아 재도전
🔥 Fever 시스템
3연속 정답 → FEVER 발동 x2 배율
6연속 정답 → SUPER FEVER x3 배율
오답 시 연속 카운트 및 Fever 즉시 리셋
⏲ 타임어택 점수 계산
기본 점수 20점 (일반 모드의 2배)
Fever 배율 적용 → 최대 60점/문제
타임 보너스: 남은 초 × 2점 추가
예시: 7초 남기고 정답 + FEVER = 40 + 14 = 54점
📝 문제 유형
4지선다 — 4개 보기 중 정답 선택
OX퀴즈 — 맞다/틀리다 선택
주관식 — 정답 직접 입력 (대소문자/공백 무시)
📖 오답노트
틀린 문제 자동 저장, 정답 맞추면 자동 제거
로그인 시 서버에 동기화 — 다른 기기에서도 유지
비로그인 시 해당 기기에만 저장 (로그인하면 서버 데이터 우선)
섹션별 또는 전체 초기화 가능
🏆 랭킹
전체 풀기와 타임어택 전체 모드만 반영
총점(Fever + 타임 보너스 포함) 기준 순위 산정
로그인 시 자동 기록, 유저별 최고 점수만 표시
