## XAG 접근성 퀴즈 - 구현 기능 전체 정리

---

### 1. 핵심 시스템

| 기능 | 설명 |
|------|------|
| **Google 로그인** | Firebase Auth 기반, 팝업 방식으로 PC/모바일 모두 지원 |
| **오답노트 동기화** | 서버 우선 정책 — 로그인 시 서버 데이터 로드, 세션 중 실시간 동기화 |
| **오답노트 관리** | 틀린 문제 자동 저장, 맞추면 자동 제거, 섹션별/전체 초기화 가능 |
| **Fever 시스템** | 3연속 정답 → FEVER(x2), 6연속 → SUPER FEVER(x3), 오답 시 리셋 |
| **랭킹 시스템** | Firestore 기반, totalScore(Fever 보너스 포함)로 순위 산정, 전체풀기+타임어택 반영 |
| **문제 타입 확장** | 4지선다(MC) + OX퀴즈 + 주관식(SA) 3종 지원 |
| **가중치 랜덤 출제** | 타입별 균등 배분(MC/OX/SA 3~4개씩) + 오답 문제 3배 가중치 |
| **타임어택 모드** | 문제당 10초 제한, 배점 2배(기본 20점), 남은 시간 보너스(초당 2점) |
| **게임 가이드** | 타이틀 화면에서 모달 팝업으로 모드/Fever/타임어택/문제유형/오답노트/랭킹 규칙 안내 |

---

### 2. 문제 데이터

| 항목 | 내용 |
|------|------|
| **총 문항 수** | 133문제 (MC 100 + OX 20 + SA 13) |
| **데이터 원본** | JS 파일이 원본, 엑셀은 백업 |
| **주관식 답안** | 복수 정답 허용 (accepted_answers), 공백/대소문자 무시 매칭 |

### 3. 게임 모드

| 모드 | 설명 |
|------|------|
| **랜덤 10문제** | 타입별 균등 배분 + 오답 가중치 적용, 랭킹 미반영 |
| **전체 풀기** | 선택 섹션의 모든 문제, 랭킹 반영 |
| **타임어택 랜덤 10문제** | 문제당 10초 제한, 배점 x2, 랭킹 미반영 |
| **타임어택 전체 풀기** | 문제당 10초 제한, 배점 x2, 랭킹 반영 |
| **오답노트 풀기** | 틀린 문제만 재도전, 섹션별/전체 선택 가능 |

---

### 4. 타임어택 시스템

| 기능 | 설명 |
|------|------|
| **타이머 바** | 상단에 10초 게이지가 줄어드는 바, 5초 이하 주황, 3초 이하 빨강 깜빡임 |
| **시계 아이콘 shake** | 5초 이하 흔들림 시작, 3초 이하 격렬한 흔들림 + 확대, 정답/시간초과 시 멈춤 |
| **시간초과 오답 처리** | 10초 내 미응답 시 자동 오답 + 정답 표시 |
| **타이머 정지 연출** | 정답 시 바가 초록 번쩍 + 멈춤, 오답 시 빨간색 멈춤 |
| **타임 보너스** | 정답 시 남은 초 × 2점 추가, "TIME BONUS +14" 팝업 (2초간 표시) |
| **배점 2배** | 기본 20점 × 피버 배수 + 타임 보너스 — 최대 60점+보너스/문제 |

---

### 5. 게임 플레이 이펙트

| 기능 | 설명 |
|------|------|
| **실시간 점수 표시** | 프로그레스 바 아래에 현재 총점 + 정답률 실시간 표시 (글래스모피즘 카드) |
| **정답 미니 컨페티** | 매 정답마다 작은 색종이 12개 터짐 |
| **오답 화면 흔들림** | 틀리면 화면 전체가 좌우로 흔들리는 shake 효과 |
| **오답 붉은 비네팅** | 틀리면 화면 가장자리가 붉게 번쩍 |
| **정답/오답 화면 플래시** | 정답=초록, 오답=빨간 방사형 플래시 |
| **콤보 카운터** | 3, 5, 7, 10, 15, 20 연속 정답 시 "N COMBO!" 팝업 (단계별 색상 변화) |
| **점수 팝업** | 정답 시 획득 점수가 떠올라 사라지는 효과 |
| **피버 팝업** | 피버 진입 시 "FEVER! x2" / "SUPER FEVER! x3" 중앙 팝업 |
| **피버 파티클 폭발** | 피버 진입 시 화면 중앙에서 파티클이 방사형으로 터져나감 |
| **피버 배경 맥동** | 피버=주황, 슈퍼피버=빨강 배경이 은은하게 맥동 |
| **피버 연동 파티클** | 피버 상태에 따라 배경 파티클 색상/속도 변화 |
| **결과 화면 컨페티** | 정답률 80% 이상 시 대형 색종이 효과 (만점=80개, 80%+=40개) |

### 6. UI / 비주얼 디자인

| 기능 | 설명 |
|------|------|
| **다크/라이트 모드** | 우측 상단 토글 버튼, localStorage에 설정 저장, 파티클/카드 등 전체 대응 |
| **그라데이션 배경** | 다크 네이비/인디고 그라데이션 (단색 검정 대비 깊이감) |
| **배경 파티클 시스템** | 50개 파티클 + 근접 파티클 간 연결선, Canvas 기반 실시간 렌더링 |
| **글래스모피즘** | 문제 카드, 보기 버튼, 섹션 버튼, 실시간 점수 바에 반투명 유리 효과 |
| **버튼 리플 이펙트** | 클릭 시 물결이 퍼져나가는 효과 |
| **버튼 호버 글로우** | 마우스 오버 시 파란 발광 + 떠오르는 효과 |
| **타이틀 순차 페이드인** | 제목 → 부제 → 통계 → 로그인 → 시작 → 서브 버튼 순서로 등장 |
| **시작 버튼 맥동** | 주기적으로 글로우가 커졌다 작아지는 pulse 효과 |
| **타이틀 아이콘 플로팅** | 아이콘이 위아래로 부드럽게 떠다니는 애니메이션 |
| **타이틀 버튼 레이아웃** | 시작하기(메인) + 랭킹/가이드(서브 가로 배치) |
| **섹션 버튼 stagger** | 리스트 항목이 시간차를 두고 순서대로 슬라이드인 |
| **문제 슬라이드 전환** | 다음 문제로 넘어갈 때 슬라이드 애니메이션 |
| **해설 슬라이드인** | 해설이 위에서 부드럽게 내려오며 등장 |
| **Q번호 칩 스타일** | 문제 번호가 파란 반투명 배경의 뱃지 형태 |
| **프로그레스 바 글로우** | 진행률 바에 파란 발광 효과 |
| **다음 문제 버튼** | 그라데이션 배경 + 호버 시 떠오르는 효과 |
| **게임 가이드 모달** | 페이드인 + 슬라이드업 등장, 배경 클릭/X버튼 닫기, 스크롤 위치 초기화 |

### 7. 결과 화면

| 기능 | 설명 |
|------|------|
| **등급 뱃지** | S(만점) / A(90%+) / B(70%+) / C(50%+) / D — 회전하며 바운스 등장 + 빛 폭발 |
| **S등급 금빛 맥동** | 만점 시 뱃지가 지속적으로 금색 글로우 애니메이션 |
| **점수 카운트업** | 정답 수와 총점이 0부터 올라가는 애니메이션 (ease-out) |
| **스코어 링** | SVG 원형 게이지가 정답률에 맞춰 채워지는 애니메이션 + 글로우 |
| **Fever/타임어택 보너스 표시** | 기본 점수 + Fever 보너스 + 타임어택 배점 내역 표시 |
| **결과 화면 진입** | scale 0.9 → 1로 확대되며 등장 |
